🏟️ Semifinals Pick’em

Pick the winner of each Semifinal and predict the final score, which includes extra time if played.
Correct winner = 3 points. Exact final score = 2 points.
Extra time / penalties correct = +1 point; incorrect = -1 point.

Please note: Exact score points require the correct winner pick.

For final scores, do not include penalty shootout goals. Scores must be winner-first, like 2-1, 3-0, or 1-1. Each match locks at kickoff.
